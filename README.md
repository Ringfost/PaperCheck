###接口设计与实现
####文件读取接口
1.从命令行中接受原文,查重论文的存储路径;
2.再对读取的文件进行分句处理;
3.将处理后的文件进入下一个接口。
算法：
![](https://img2020.cnblogs.com/blog/2532113/202109/2532113-20210919185014563-1243339434.png)

####计算模组接口
1.读取由上一个接口交付的两个字符数组；
2.对查重论文中的所有句子在原文中进行比对并取最高值。
